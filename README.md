# タイピングサバイバー（Unity）

## 概要

タイピングサバイバーは、タイピング×サバイバル2Dアクションをテーマにしたゲームである。プレイヤーは洞窟内のブロックをタイピングによって破壊し、洞窟内に点在するアイテムを駆使しながら、できるだけ長く生存することが目的となっている。

---

## デモ

![demo](demo/play.gif)

---

## ゲーム内容

* タイピングによってブロックを破壊
* ランダム生成されるマップとアイテム
* HPが徐々に減少するサバイバル要素
* アイテム（回復・破壊など）による戦略性
* 2人対戦モード（妨害要素あり）

## 工夫した点

* タイピングとアクションゲームの融合
* ランダム生成によるリプレイ性の向上
* タイピングスキルだけでなく戦略性も求められる設計
* UI・画面遷移の設計による操作性の向上

---

## 制作意図

タイピング練習にゲーム性を持たせ、楽しみながらスキル向上できる体験を目指した。
ランダム性やアイテム要素を取り入れることで、単純なスキル勝負に偏らないゲーム設計とした。
